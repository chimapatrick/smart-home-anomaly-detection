## Overview
This project demonstrates a lightweight Artificial Intelligence (AI) approach for detecting anomalous behaviour in smart home IoT environments.

Smart homes rely on interconnected devices that continuously exchange data. This connectivity introduces cybersecurity risks such as unauthorized access and abnormal device activity. This project uses a machine learning anomaly detection model to identify suspicious behaviour patterns.

## Objective
To simulate real-time monitoring of IoT device behaviour using a lightweight AI model suitable for resource-constrained smart home environments.

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Matplotlib

## Method
The system uses an **Isolation Forest** algorithm to:
- Learn normal device behaviour
- Detect anomalies in network activity
- Flag suspicious events

## Dataset
A simulated smart home IoT dataset containing:
- Device traffic frequency
- Packet size
- Connection duration
